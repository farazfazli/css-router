CSS Router allows you to create single page websites that swap out views, without having to use JavaScript.

Getting Started

Thanks for wanting to add CSS Router to your website! First, download the template from the GitHub repo, see here. After downloading, you should be ready to get started!

1. Replace the navbar text. 

      
        <div class="navbar">
        <label for="one"><strong>CSS Router</strong></label>
        <label for="two">Getting Started</label>
        <label for="three">Motivation</label>
        <label for="four">Contact</label>
        <hr>
        </div>

2. Replace the content in each menu section class. There are four included by default: one, two, three, and four. The menu section class number cooresponds with the label for, so when the label is clicked, it displays the cooresponding content.

3. That's all, I hope you enjoy using CSS Router! If you have any questions or need help, don't hesitate to open an issue on the GitHub Repo! And don't forget to star the repo if you enjoy this project!

Motivation

I created CSS Router because I saw all of the different JavaScript libraries for doing this, and thought of all the people who do not have JavaScript enabled who are not able to have the same user experience. There are many NoScript users, and users of the Tor browser who do not use JavaScript at all, and not many keep those people in mind when designing their website.
