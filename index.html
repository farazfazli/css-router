Motivation

I created CSS Router because I saw all of the different JavaScript libraries for doing this, and thought of all the people who do not have JavaScript enabled who are not able to have the same user experience. There are many NoScript users, and users of the Tor browser who do not use JavaScript at all, and not many keep those people in mind when designing their website.
